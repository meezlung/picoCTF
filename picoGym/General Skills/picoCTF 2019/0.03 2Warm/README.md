## Description
Can you convert the number 42 (base 10) to binary (base 2)? 

## Approach
Conversion from base 10 to base 2 can be done by dividing the number by 2 until the quotient is zero. 

```
# Number divides 2   Quotient   Remainder
1 42                 21         0
2 21                 10         1
3 10                 5          0
4 5                  2          1
5 2                  1          0
6 1                  0          1
```
